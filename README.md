#REMASTER WIN-LOCK by @coult, Creator: @Me_C0DER for Telegram, password to the archive: tgk:@coult
A simple Python utility built using the `tkinter` library that creates a full-screen lock window featuring ASCII art, a countdown timer, and a password entry field. This project was developed strictly for demonstration, educational, and entertainment purposes (pranks/jokes).

🚀 Features
- Full-Screen Mode: The application expands to fill the entire screen, overlaying the desktop.
- Customization: Easily change the password, main message text, and labels directly within the code.
- ASCII Art: Features a stylish built-in skull on the left side of the screen.
- Countdown Timer: A visual timer (set to 3 hours and 30 minutes by default) to add interactivity.
- Automatic Verification: The application automatically closes 2 seconds after the correct password is typed (no need to click the button).

🛠 Configuration
All project configurations are managed inside the `REMASTER WIN-LOCK by @coult.py` file:

Change Password: Locate the `PASSWORD = "123123"` line.
Main Text: Locate the `text="asdasd"` line inside the `title_label` block (e.g., change to "Windows is blocked").
Footer Subtitle: Locate the `text="blablabla"` line inside the `by_label` block (e.g., change to "by @coult").
